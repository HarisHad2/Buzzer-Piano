# Buzzer Piano

This project is a Arduino project were I have made a buzzer play a note by the push of a button. 

## Material

 * Arduino Uno
 * Breadbord
 * 4 Jumper Wires
 * 1 Button 
 * 1 Resistor 1k ohm
 * 1 Buzzer
